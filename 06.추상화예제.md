#### 추상화 미적용 코드 정리

1. 요구 사항이 추가될수록 코드가 길어지고 구조는 복잡해진다.
    - 새로운 클라우드 추가시 모든 메서드에 새로운 if 블록을 추가해야한다.

2. 관련 코드가 여러 곳에 분산된다.
    - 하나의 클라우드 처리와 관련된 코드가 여러 메서드에 흩어진다.

결과적으로 코드 가독성과 분석 속도가 저하를 야기시킨다.
또한 코드 추가에 따른 노동 시간이 증가하며, 실수하기 쉬워 불필요한 디버깅 시간이 증가할 것이다.
